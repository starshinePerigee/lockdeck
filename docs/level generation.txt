---

build "depth templates"

a depth template is a set of depths, level building instructions (with unique depth counts per number of pins), a category (bonus, complication, danger), a net hazard, and a set of weights (early: 1-3 pins, mid: 4-5 pins, late: 5 pin extended)

starting from danger, draw depths based on weight until a hazard threshold is reached. (maybe draw one hazard, one complication, and then just draw from a random category to allow lots of complications and a few hazards or a hazard dense but otherwise simple lockpl)

then place all the depths according to their rules. track which depths were placed successfully
