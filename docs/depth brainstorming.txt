- jam break: if you hit it, and have pins jammed, it breaks the picks that applied the jam
- jam limit: pins have a fixed, random max amount of jam. specific depths can reveal / disable this limit
- test break: if you test this depth (twice?) it breaks the test pick (scans as safe / scans as dangerous)
- test limit: upper bound on number of test on a pin at once / total number of tests on a segment / total number of tests all time
- rush: if you test / reveal a depth, you have to hit a (different depth/the revealed depth/set the pin) (next turn / in n turns) or the pick breaks (could reveal itself on test)

- pair: if you test two specific depths at once, the testing pick breaks
- link: the scanning pick is linked and breaks if you break the next pick / any pick

- jam trap: if you jam the pin past the trap depth, it breaks the jam pick
- crush trap: reads as safe, if you try to crush it it breaks the crush
- minima: if you test this depth, it pulls you to it
- generic: does (bad effect) if you (test / reveal / jam) (without passing / activating this depth)
- safe crush: crushes backwards, but kicks you backwards
- darkness: if you hit both, the lights go out
- random break: breaks from discard
